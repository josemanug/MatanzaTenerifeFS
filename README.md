# MatanzaTenerifeFS
Aplicación para el control de inventario de equipaciones en el almacén y jugadores fichados.

En este repositorio se irán subiendo los avances para la obtención de los objetivos.


## Stack Tecnológico

Esta aplicación se compone de un backend en Java con Spring Boot para el desarrollo de una API que controlará las comunicaciones con una base de datos en MySQL.

Por otro lado, esta API será consumida por un cliente Frontend mediante React JS para el mostreo del contenido alamcenado en la base de datos.

## Objetivo

El principal objeto de este proyecto es el desarrollo completo de un sistema de gestión de inventario, en este caso concreto, del Matanza Tenerife de fútbol sala de isla de Tenerife, con el propósito de poder saber el stock de equipaciones en el almacén junto con datos del jugador como su nombre, dorsal que porta o la categoría en la que juega.
